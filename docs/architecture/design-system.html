🎨 Visual Design System

Creating a consistent and beautiful experience for YSAs across Europe

Design Philosophy

RisingGen's design system is built around the principle of Unity in Faith. Just as YSAs across Europe are united in their beliefs while celebrating diverse cultures, our design creates visual harmony while allowing for cultural expressions.

🎯 Design Goals

  • Inspiring: Elevate the spiritual experience of gathering
  • Inclusive: Welcome YSAs from all European backgrounds
  • Trustworthy: Reflect Church standards and values
  • Accessible: Usable by everyone, regardless of ability

Accessibility Standards

Following Church principles of inclusion, our design ensures everyone can participate fully in the YSA community.

♿ Accessibility Features

  • Color Contrast: All text meets WCAG AA standards (4.5:1 ratio minimum)
  • Font Size: Minimum 16px for body text, scalable up to 200%
  • Keyboard Navigation: All interactive elements accessible via keyboard
  • Screen Readers: Proper semantic markup and alternative text
  • Focus Indicators: Clear visual focus states for navigation
  • Language Support: Compatible with European language variations
