MongoDB Setup
To Do

Local Installation

# MongoDB connection string for development mongodb://localhost:27017/risinggen-dev # Start MongoDB service brew services start mongodb-community # macOS sudo systemctl start mongod # Linux net start MongoDB # Windows

Docker Alternative

# Run MongoDB in Docker docker run -d \ --name risinggen-mongo \ -p 27017:27017 \ -e MONGO_INITDB_DATABASE=risinggen-dev \ mongo:5.0
PostgreSQL Setup
To Do

Local Installation

# PostgreSQL connection for events data postgresql://localhost:5432/risinggen_events # Create database and user createdb risinggen_events createuser risinggen_dev -P

Docker Alternative

# Run PostgreSQL in Docker docker run -d \ --name risinggen-postgres \ -p 5432:5432 \ -e POSTGRES_DB=risinggen_events \ -e POSTGRES_USER=risinggen_dev \ -e POSTGRES_PASSWORD=development \ postgres:14

Start Development Environment
To Do

All Services at Once

# Start all development services npm run dev:all # This will start: # - Frontend React app on http://localhost:3000 # - Backend API server on http://localhost:3001 # - Database services (if using Docker)

Individual Services

# Start services individually npm run dev:frontend # React app on :3000 npm run dev:backend # API server on :3001 npm run dev:db # Database services # Or use PM2 for process management pm2 start ecosystem.config.js

üîß Troubleshooting

Common Issues

Port Already in Use
# Kill process on port 3000 lsof -ti:3000 | xargs kill -9 # Or specify different ports PORT=3002 npm run dev:frontend API_PORT=3003 npm run dev:backend
Database Connection Issues
# Test MongoDB connection mongo --eval "db.adminCommand('ismaster')" # Test PostgreSQL connection psql -h localhost -p 5432 -U risinggen_dev -d risinggen_events
Node Modules Issues
# Clean install rm -rf node_modules package-lock.json npm install # Clear npm cache npm cache clean --force
